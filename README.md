# Capital Gains Tax Calculator USA 2025-2026

Free online calculator to estimate capital gains tax on stocks, crypto, real estate, and other investments.

## Features

- Calculate short-term and long-term capital gains tax
- 2025-2026 tax rates (updated for inflation)
- All filing statuses supported
- Mobile responsive design
- SEO optimized

## Deployment on Vercel

1. Push this code to your GitHub repository
2. Go to [vercel.com](https://vercel.com)
3. Click "Add New Project"
4. Import your GitHub repository
5. Click "Deploy"

That's it! Your website will be live in 2-3 minutes.

## Tech Stack

- Next.js 15
- React
- Tailwind CSS
- shadcn/ui
